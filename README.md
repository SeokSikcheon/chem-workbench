# Chem Workbench（化学工作台）

An interactive chemistry workbench for visualizing atoms and the periodic table.  
一个用于可视化原子结构与元素周期表的交互式化学工作台。

🔗 Online Demo / 在线演示：  
https://seoksikcheon.github.io/chem-workbench/

---

## ✨ 项目简介 | Introduction

**Chem Workbench** 是一个面向学习与演示的化学可视化工具，  
旨在通过直观的交互方式，帮助理解：

- 原子结构
- 电子排布
- 电子云示意
- 轨道形状（s / p / d / f）
- 壳层结构与价层概念
- 元素周期表中的位置与分类

This project focuses on **visual learning** of atomic structure and the periodic table,  
making abstract concepts easier to understand through interactive graphics.

---

## 🧪 主要功能 | Features

- 🧬 原子序数（Z）调节与对应元素切换  
- 📊 自动生成电子排布（按构造原理顺序）  
- 🌫️ 电子云可视化显示  
- 🧱 轨道示意（s / p / d / f）叠加展示  
- 🪐 壳层结构（K, L, M, N, ...）可视化  
- 🧭 元素周期表联动高亮  
- 📏 共价半径趋势可视化模式  
- ☢️ 放射性元素标记与提示  
- 📚 分类说明（如碱金属、卤素、稀有气体等）  
- 🌗 支持深色界面风格（适合课堂演示与投屏）

## 🚀 如何使用 | How to Use

1. 打开在线演示页面：  
   👉 https://seoksikcheon.github.io/chem-workbench/

2. 通过滑块或按钮调整原子序数  
3. 观察电子排布、电子云、轨道与壳层结构的变化  
4. 在周期表中查看对应元素的位置与分类信息
